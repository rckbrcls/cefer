- `admin@usp.br` always receives the `Administrador` assignment.
- `funcionario@usp.br` receives a random non-admin assignment.
- `interno@usp.br` is an internal USP user and is not staff.
- `cadastro@usp.br` is intended for registration-flow testing:
  - It is created as `PESSOA` and `INTERNO_USP`.
  - It does not have an active `USUARIO_SENHA` account.
  - Use CPF `01995923222` and NUSP `12345681` to test registration requests.
